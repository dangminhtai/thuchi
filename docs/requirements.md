# Yêu cầu dự án (Requirements)

- REQ001: Xây dựng ứng dụng quản lý thu chi Android hoàn chỉnh dựa trên hình ảnh cung cấp.
    - Chạy được trên Android Studio hiện tại.
    - Ngôn ngữ: Tiếng Việt.
    - Tính năng chính: Nhập thu chi, Báo cáo biểu đồ, Xem lịch giao dịch.
    - Tuân thủ nguyên tắc SOLID.
    - Cấu trúc dự án rõ ràng, chia module/package hợp lý.
