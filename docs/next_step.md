# Bước tiếp theo (Next Steps)

1. **Đồng bộ hóa đám mây**: Hỗ trợ lưu trữ dữ liệu trên Firebase hoặc Google Drive để tránh mất dữ liệu khi đổi thiết bị.
2. **Cài đặt mật khẩu**: Thêm tính năng khóa ứng dụng bằng mã PIN hoặc vân tay (như trong ảnh mẫu có phần bảo mật).
3. **Quản lý ngân sách**: Thiết lập hạn mức chi tiêu cho từng danh mục và thông báo khi sắp vượt mức.
4. **Xuất báo cáo**: Hỗ trợ xuất dữ liệu ra file Excel hoặc PDF.
5. **Đa ngôn ngữ**: Mở rộng hỗ trợ i18n nếu cần thiết trong tương lai.
